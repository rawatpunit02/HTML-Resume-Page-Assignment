## Quick Setup

Prerequisites:
- Any modern web browser (Chrome, Firefox, Edge, Safari)

Steps:
1. Open a terminal and change to the project folder:
  ```powershell
  cd "c:\\Cohort2026\\HTML assignment"
  ```
2. Open the page directly by double-clicking `resumePageAssignment.html` or run a simple local server:
  ```powershell

  # Then open:
  http://localhost:8000/resumePageAssignment.html
  ```

That's it — the page will load in your browser.
